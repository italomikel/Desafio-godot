# Desafio Godot

Este é um repositório para o Bootcamp da Dio, onde criei um jogo com a Godot Engine. Nele, foram desenvolvidos:

* Movimentação
* Ataque
* Animação
* Cenário
* Inteligência artificial para o mob seguir o jogador
* Barra de vida
* Dano
* Interface de usuário
* Tela de game over
